Trabajo final Robótica Industrial 
El objetivo principal del trabajo era realizar un manual de usuario el cual tuviera en su contenido los aspectos más importantes del robot, y un ejemplo de código donde se pudiera mover el robot. 
1. Se realizó un manual de usuario explicando cada una de las partes del robot, sus configuraciones, encendido y apagado,
la creación de un programa nuevo, edición y ejecución. 
2. En la segunda parte del manual se pone el paso a paso de lo que se hizo en el laboratorio y cómo se hizo un movimiento 
con el robot FANUC ARC Mate 100i, junto a un ejemplo de código el cual fue creado en el laboratorio, y el robot se movió exitosamente. 
3. En el archivo .txt llamado CÓDIGO_MOVIMIENTO se puede ver el ejemplo de código usado para mover el robot.
